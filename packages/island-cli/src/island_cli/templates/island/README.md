# {{game}} Island Package

An Archipelago randomizer implementation for {{game}}.

## Installation

1. Download the latest `.island` file from releases
2. Place it in your Archipelago `worlds/` directory
3. Run Archipelago

## Configuration

See the [setup guide](docs/setup_en.md) for detailed configuration options.

## Development

### Building

```bash
island build
```

### Testing

```bash
pytest tests/
```
